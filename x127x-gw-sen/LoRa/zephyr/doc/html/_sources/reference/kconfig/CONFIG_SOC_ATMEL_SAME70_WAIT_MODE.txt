.. title:: SOC_ATMEL_SAME70_WAIT_MODE

.. option:: CONFIG_SOC_ATMEL_SAME70_WAIT_MODE:
.. _CONFIG_SOC_ATMEL_SAME70_WAIT_MODE:

For JTAG debugging CPU clock (HCLK) should not stop. In order
to achieve this, make CPU go to Wait mode instead of Sleep
mode while using external crystal oscillator for main clock.



:Symbol:           SOC_ATMEL_SAME70_WAIT_MODE
:Type:             bool
:Value:            "n"
:User value:       (no user value)
:Visibility:       "n"
:Is choice item:   false
:Is defined:       true
:Is from env.:     false
:Is special:       false
:Prompts:

:Default values:

 *  y (value: "y")
 *   Condition: DEBUG && SOC_ATMEL_SAME70_EXT_MAINCK (value: "n")
