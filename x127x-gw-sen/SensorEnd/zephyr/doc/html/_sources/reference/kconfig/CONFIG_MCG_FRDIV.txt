.. title:: MCG_FRDIV

.. option:: CONFIG_MCG_FRDIV:
.. _CONFIG_MCG_FRDIV:

Selects the amount to divide down the external reference clock for the
FLL. The resulting frequency must be in the range 31.25 kHz to 39.0625
kHz.
