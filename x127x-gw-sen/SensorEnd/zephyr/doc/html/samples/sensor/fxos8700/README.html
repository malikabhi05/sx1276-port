FXOS8700 Accelerometer/Magnetometer Sensor¶

Overview¶

A sensor application that demonstrates how to use the fxos8700 data ready interrupt to read accelerometer/magnetometer data synchronously.

Building and Running¶

This project outputs sensor data to the console. It requires an fxos8700 sensor, which is present on the frdm_k64f, frdm_kw41z, and hexiwear_k64 boards. It does not work on QEMU.

$ cd samples/sensors/fxos8700
$ make BOARD=frdm_k64f
